# 🛒 E-Commerce Sales Analytics Dashboard

## 📌 Overview
This project analyzes e-commerce sales data to uncover key insights into revenue trends, product performance, and customer behavior. The results are presented through an interactive Power BI dashboard.

---

## 🎯 Business Objective
- Identify top-performing products and categories
- Analyze sales trends over time
- Evaluate regional performance
- Provide actionable recommendations to improve revenue

---

## 🛠 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (MySQL / PostgreSQL)
- Power BI
- Excel

---

## 📊 Dataset Description
The dataset contains:
- Order Date
- Product Category
- Sales Amount
- Quantity Sold
- Customer Details
- Region

(Sourced from Kaggle sales dataset)

---

## 🔎 Analysis Performed

### 1. Data Cleaning
- Removed null values
- Standardized column formats
- Removed duplicates

### 2. Exploratory Data Analysis (EDA)
- Monthly revenue trends
- Top 10 products by sales
- Region-wise performance

### 3. SQL Analysis
- Total revenue calculation
- Customer segmentation
- KPI queries (growth, average order value)

### 4. Dashboard (Power BI)
- Revenue trends over time
- Sales by region
- Top-performing products
- KPI cards (Total Sales, Growth Rate)

---

## 📈 Key Insights
- Identified highest revenue generating product category
- Found seasonal spikes in sales
- Pinpointed top-performing regions contributing to business growth
